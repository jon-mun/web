### Overview

A mobile app that provides personalized food and exercise recommendations using machine learning. Built as part of the Bangkit Academy capstone project.

### What I Did

- Designed and implemented the backend using NestJS.
- Built authentication using custom JWT.
- Integrated and deployed the ML model using Flask on Cloud Run.
- Managed CI/CD pipelines and automated Docker image builds.
- Configured Cloud SQL and automated DB migrations.

### Challenges

- Coordinating between the mobile, AI, and backend teams.
- Ensuring consistent communication between multiple services across GCP.
- Delivering a fully functional MVP under a strict one-month timeline.
