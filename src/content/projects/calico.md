### Overview

A mental-health-focused chatbot that provides supportive responses and guided conversations. Built for the Google Solution Challenge 2023.

### What I Did

- Implemented mobile features in Flutter.
- Integrated Firebase for database and user authentication.
- Built the chatbot integration through REST APIs combining GPT-3 and Dialogflow.
- Collaborated with UI/UX and backend engineers to ship a polished app.

### Challenges

- Balancing natural language capabilities with privacy and scope limitations.
- Managing rapid prototyping while maintaining a consistent mobile experience.
