### Overview

A platform for the 2023 Joints Logic Competition. The site provides event information and acts as the official exam system, automatically saving and grading participant answers.

### What I Did

- Managed the entire development team (backend + frontend).
- Designed the system architecture and database schema.
- Built authentication and exam-grading logic using TRPC.
- Deployed services to Vercel and MongoDB Atlas.
- Set up an admin panel using Retool.

### Challenges

- Tight one-month development window.
- Ensuring reliability during live exam periods.
