### Overview

A voting system for electing the 2022 Head of Himakom. Built to replace the previous manual process with a trusted online system.

### What I Did

- Designed the backend API and database structure.
- Integrated backend services with the React frontend.
- Deployed the application to Heroku and MongoDB Atlas.

### Challenges

- First full-stack project after self-learning, requiring rapid upskilling.
- Ensuring data integrity and avoiding vote duplication.
